¿Por qué fue útil usar GitHub?

  • GitHub Pages permite publicar el sitio gratis y de forma automática desde el repositorio.
  • Control de versiones para ver cambios, volver atrás y trabajar sin perder historial.
  • Colaboración mediante ramas y pull requests para revisar aportes.
  • Issues y proyectos para organizar tareas y errores.
